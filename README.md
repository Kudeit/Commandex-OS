Hisaki Commandex is a command-line interface (CUI) OS similar to MS-DOS, specifically designed for UEFI 64-bit systems.
This OS is very lightweight and runs on a UEFI AMD64 PC environment.
We are also considering porting stdio.h and the TCC compiler in the future, which will allow for application development using simple C language.
In the future, it may be possible to operate with 32MB of memory.
This OS is in ISO format. During prototype verification, it was burned to a Rufus disc and run on a Surface Pro 9, confirming its operation. It also works with VMware and VirtualBox.
